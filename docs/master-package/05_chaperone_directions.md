## 1. 기존 샤페론 논리 — 재서술 (손실 없이)

### 1.1 성공 조건 계층 (암브록솔 이식 체크리스트 요약)

| ID | 조건 | IDSX 상태 |
|----|------|-----------|
| 1 | 잔존 촉매 활성 | 대부분 UNKNOWN — **최대 병목** |
| 2 | 오접힘/불안정/ER 체류가 주원인 | 구조 프록시 부분 충족 |
| 3 | null·촉매 kill 제외 | 규칙 v1로 충족 가능 |
| 4 | 가역 결합 | 미충족 |
| 5 | ER 안정 / 리소좀 해리 또는 기질 교체 | 미충족 |
| 6 | 영구 활성부위 차단 금지 | 6.5Å 규칙으로 부분 |
| 7 | 리소좀 국소화↑ | 미충족 |
| 8 | genotype-specific | TIER 프록시 부분 |
| 12–13 | 구조 설명 + 실패 모드 공개 | 부분–인식 |

**1편에 충분한 최소 세트:** 3, 6, 8, 12, 13.  
**PC 제안 논문:** +1, 4, 5, 7.

### 1.2 모달리티 삼분법 (유지)

```
EXCLUDE ──► 비구제
CAUTION_POCKET ──► 소분자 inhibitory PC만 (잔존활성 전제)
TIER1/2 ──► 표면 바인더 / 알로스테릭 / (에피토프 부족 시) 재설계
```

### 1.3 변이 자리 ≠ 결합 자리 (불변 공리)

- Ser152 RSA 0% → 직접 결합 논리 **폐기**  
- 에피토프: Y151, P150, S154…  
- RFdiffusion `hotspot_res`는 **표면 앵커**이지 152 강제 지정 아님

### 2.1 Amenability 전량 (Prototype v1.0)

| 티어 | n (변이) |
|------|----------|
| EXCLUDE | 210 |
| CAUTION_POCKET | 106 |
| TIER1 | 146 |

| TIER3 | 6 |

- 점수 부여 행 ~396, 평균 점수 ~6.2, 최대 10  
- 위치 단위 hotspot 시트 ~92 위치  
- **동일 위치에 티어 혼재** 다수 (null vs missense 등) → collapse 규칙 필수

### 2.2 EvoEF2 ΔΔG (full_ddg_results, 643행)

- 상위 물리 불안정 예: G336W(+214), D478Y(+207), C422 계열(+120대), C184W(+111)  
- S152I 계열은 중간 고에너지(+46 전후, 별도 요약표)  
- **데이터 품질 경보:** P451R/L 등이 ddG≈451로 기록 — **잔기 번호가 에너지에 샌 이상치 가능성 큼**  
  → 재분석·클리닝 없이 Track B 랭킹 확정 금지

### 2.3 CTSS (591행)

상위 예: H159Y, C422Y/F, D478Y, G55V, P261L, S149Y…  
SASA=0 & 고ΔΔG: S333W, G312D, C184W, **D308E** 등 → “에너지는 터지는데 틈이 없음”

### 2.4 방법론 렌즈 정리 (모순이 아님)

| 렌즈 | 질문 | 우승 예 |
|------|------|---------|
| 임상·적합성 | 누구에게 구제 **논리**? | Ser152, Asp308 |
| 열역학 붕괴 | 무엇이 가장 불안정? | C184, C422, G336… |
| 소분자 접근성 | 어디에 약이 붙을 틈? | C422, H159Y… |

**신규 방향의 핵심은 렌즈를 섞지 않는 운영체제.**

## 3. 샤페론 분석의 “빈 칸” — 지금 확장해야 할 축

### 3.1 Residual activity graph (최우선)

TIER1 146을 전부 실험할 수 없다. 필요한 것:

1. 문헌 기반 **잔존활성 있음 / 없음 / 불명** 3색 맵  
2. 표현형 (신경형 vs attenuated) 병기  
3. 같은 코돈 다중 치환 비교 (S152I vs R vs N; D308G vs E vs N)

**산출물 이름 제안:** `Residual_Activity_Literature_Matrix_v1.xlsx`

### 3.2 pH-split 설계 명세 (소분자 트랙)

| 항목 | 명세 |
|------|------|
| ER pH 가정 | ~7.2–7.4 |
| Lysosome pH | ~4.5–5.0 |
| 이상적 PC | 중성에서 \(K_d\) 낮음, 산성에서 유효 결합 약화 또는 기질 \(K_m\) 경쟁 가능 |
| 바인더 | pH 안정 + 프로테아제 저항 별도 엔지니어링 |

문서화만으로도 자문 설득력이 오른다 (실험 전 **설계 규격서**).

### 3.3 에피토프 품질 스코어 (개수 너머)

현재: RSA≥25% 잔기 개수.  
확장:

| 품질 축 | 이유 |
|---------|------|
| 오목도 / pocketness | RFdiffusion 성공률 |
| 소수성 패치 면적 | 결합 구동력 |
| 전하 상보 가능 잔기 | Asp308 전략 |
| 진화 보존 표면 | 오프타겟·면역 |
| 글리코 근접 | 입체 장애 |
| 촉매 6.5Å 침범 여부 | 안전 |

**산출물:** `Epitope_Quality_Card_Ser152_Asp308_Tyr108.md`

### 3.4 실패 모드 라이브러리 (D2S0 교훈 일반화)

| 실패 ID | 현상 | 방지 규칙 |
|---------|------|-----------|
| F1 | docking OK, 활성 무회복 | competence KPI 강제 |
| F2 | 안정화 + 영구 억제 | pH/washout 시험 |
| F3 | 빈도 높은 촉매 인접 | CAUTION 트랙 |
| F4 | 이황화 파괴에 클램프 과신 | EXCLUDE vs Track B 화해 문서 |
| F5 | AF 유사 = 무해 | 해석 규율 |
| F6 | TIER1 = 처방 | non-claim |

### 3.5 IDS-native PC 문헌 가중치 재조정

암브록솔 비중을 낮추고:

- Hoshina 2018 (D2S0 등)

## 8. 세션 결론 — 샤페론 분석을 한 단계 올리려면

1. **잔존활성 문헌 매트릭스** 없이 설계 확장은 공허하다.  
2. **렌즈 분리 운영체제** (적합성 / 붕괴 / 접근성)가 신규 방향의 OS다.  
3. **에피토프 품질**이 다음 기술적 병목이다 (개수→품질).  
4. **화공 KPI·PFD**가 포트폴리오 차별화 축이다.  
5. **데이터 클리닝** (ΔΔG 이상치) 이 Track B의 전제다.
