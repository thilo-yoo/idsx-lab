## 3. 개념 한 장 (전체 융합)

```
[질병] MPS II / IDS 결핍
        │
        ├─ 모달리티 지형 ── ERT · BBB-ERT · HSCT · AAV/HSC-GT · 편집 · PC
        │                      (04_모달리티 덱 · HUB_02)
        │
        └─ IDSX 본선 ────── 구조 amenability triage (5FQL × 변이)
                              │
                              ├─ 화공: 수지·Da·선택도·Ki창·PFI  (03_화공 덱)
                              ├─ 시뮬: S1–S8 · STEP3 · STEP7   (02_시뮬 덱)
                              ├─ 고분자: L3 제제 GO · L1 NO-GO (05_고분자 덱)
                              └─ 기업: 아바타 = 전달/생산 직교 (부록)
```

**성공 정의:** catalytic competence (결합 occupancy 아님).  
**Tm:** L1에서는 온도계 · L3 제제에서는 목적 가능.

## 4. 지금 할 일 (단순 3줄)

1. **본선:** residual 문헌 표 + (가능 시) ΔΔG 교차  
2. **직교 (선택):** 모델 효소 PEG 열안정 (고분자 L3)
