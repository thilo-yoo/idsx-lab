## 테제

샤페론은 예쁜 결합 분자가 아니라,  
**수율(X) · 선택도(S) · 촉매 수명(a) · 운전창**을 맞추는 **공정 첨가제**다.

효소 IDS는 동시에:
- **생성물** (만들어야 할 접힌 분자 inventory)
- **촉매** (GAG를 분해하는 active site)

---

## 1. 반응공학 적용

| 개념 | 샤페론 설계 적용 |
|------|------------------|
| 직렬 반응 | U→N→수출→리소좀→GAG 분해 |
| 병렬 부반응 | ERAD, 응집, 비선택 NC/EC |
| **선택도 S** | \(K_{d,N}/K_{d,U}\), 에피토프 특이성, 6.5Å 비간섭 |
| **전환율 X** | \(\eta_{\mathrm{fold}}\), Da = k_fold·τ_ER |
| Damköhler | PC로 유효 k↑ → Da↑ → X↑ |
| 체류시간 | ER/수송 τ는 세포 고정, 설계 변수는 k |
| 억제 동역학 | competitive MM, washout |
| 목적함수 | max Y·S·a → competence |

### 네트워크 (요약)

```
번역 → U ─┬─► N ─► 수출 ─► E_lys ─► 생성물(P)
          ├─► ERAD / trap
          └─► (C) 비생산 복합체
N+C ⇌ NC   (ER 안정 저장)
E+C ⇌ EC   (억제 — Lys에서 해리 필요)
```

## 2. 촉매공학 적용

| 개념 | 샤페론 설계 적용 |
|------|------------------|
| 활성점 | FGly84/Ca 기하 — 파괴 시 EXCLUDE |
| 가역 피독 | inhibitory PC = 조건부 흡착 |
| Ki 창 | Ki(ER)≪Ki(Lys) 또는 pH 의존 = 치료 창 |
| TOF | \(k_{\mathrm{cat}}\cdot\theta_{\mathrm{free}}\) |
| 담지체 안정화 | 표면 패치 바인더 = 활성점 비점유 안정화 |
| 금속–지지체 | 알로스테릭 네트워크 전달 (γ) |
| 비활성화 \(k_d\) | ERAD·프로테아제; PC는 \(k_d\)↓ |
| 이중기능 | 안정화 + (원치 않는) 억제 동시 — 창으로 분리 |

### 두 설계 계열

| | 활성점 PC | 지지체/알로스테릭 |
|--|-----------|-------------------|
| 유추 | 활성점 흡착질 | 담지체·바인더 |
| 강점 | 접힘 안정화 사례 | TOF 보존 가능 |
| 위험 | 과억제 | 전달 약함 |
| IDSX | CAUTION 관리 | Ser152 패치 |

## 3. 설계 기호 치트시트

| 기호 | 의미 | 측정/프록시 |
|------|------|-------------|
| Y | inventory 수율 | 발현·성숙·국재 |
| S | 경로 선택도 | Kd 비, 패치 특이 |
| Da | k·τ | ΔΔG, 접힘 동역학 |
| Ki(pH) | 억제 창 | pH-DSF, washout |
| TOF | 유효 촉매 속도 | 4MU assay |
| a(t) | 수명 | 반감기, 프로테아제 |
| J | max Y·S·a | 사전등록 competence |
