# Track B Neo-pocket Critical Review v0

**지위:** 가설 트랙 비평 문서 (치료 후보 확정 아님)  
**날짜:** 2026-07-17  
**관계:** Amenability Track A와 **병합 금지**. 충돌 시 A 규칙이 1편 기본값.

---

## 1. Track B가 답하려는 질문

> “변이가 단백질 표면에 **약이 물리적으로 접근 가능한 틈(neo-pocket)** 을 만들고,  
> 그 틈을 소수성·정전기 클램프로 붙잡아 **구조 붕괴를 완화**할 수 있는가?”

Track A 질문과 다름:

> “이 미스센스에 **샤페론/표면 안정화 논리를 시도할 구조적 가치가 있는가?**  
> (변이 자리 ≠ 결합 자리 가능)”

---

## 2. 사용 중인 지표와 한계

### 2.1 CTSS (프로젝트 정의)

대략: \( f(\Delta\Delta G,\; \mathrm{SASA},\; \mathrm{hydrophobicity}) \)  
원시 Top: H159Y, C422Y/F, D478Y, G55V…

| 한계 | 설명 |
|------|------|
| Forcefield | EvoEF2 점수가 실험 ΔΔG와 1:1 아님 |
| SASA | 단일 정적 구조·변이 모델링 가정 |
| 고에너지 | 너무 불안정하면 **구제 불능**일 수도 |
| 이황화 | C422/C184는 amenability **Hard EXCLUDE** 후보 |
| 이상치 | P451 등 ddG≈position 오염 가능 → **재랭킹 전 QC 필수** |

### 2.2 “폭발 코어” 서사의 위험

고ΔΔG + 이황화 파괴를 “이상적 샤페론 타겟”으로 단정하면:

1. EXCLUDE 규칙과 모순  
2. 접힘 가능 분율 ≈ 0 인 대립유전자에 클램프를 시도하는 셈  
3. 잔존활성 전제(PC 일반 조건 #1) 위반 가능성

**수정 서사:**  
고ΔΔG 노출형은 **“구조적으로 극적인 관찰”** 이지, **“구제 용이”** 가 아니다.  
구제 용이 후보는 오히려 **중간 불안정 + 잔존 접힘 능력** 쪽일 수 있다 (PC 문헌 일반론).

## 4. QC 프로토콜 (Track B 재개 조건)

1. `full_ddg_results.csv`에서 \(|\mathrm{ddG} - \mathrm{Position}| < 1\) 또는 ddG > 300 등 **물리 비합理** 행 플래그  
2. 이황화 Cys 변이 플래그 병합  
3. ClinVar 라벨·null 공존 제거  
4. SASA 계산 재현 (동일 PDB, 동일 probe)  
5. FoldX/Rosetta로 Top20 교차  
6. 그 후 CTSS 재계산 → **CTSS_v2**

v2 나오기 전: 발표에서 “C422 최종 타겟” 문구 **사용 금지**.

## 5. 파마코포어 스케치의 올바른 지위

기존 스케치 (Arg/Glu 앵커 + 중심 방향족):

- **허용:** “구조 관찰에 기반한 **설계 가설 만화**”  
- **금지:** “합성할 최종 약물 구조”  
- **다음:** 에피토프 실측 거리 재확인, 촉매 침범 체크, 유사체 라이브러리 **가상**만
