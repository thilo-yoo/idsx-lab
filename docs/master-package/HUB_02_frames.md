## B. 고분자 (감사 후 최종판 = 26)

**층 분리**

| 층 | 내용 | 판정 |
|----|------|------|
| L1 세포 내 PC | 고분자로 대체 | **NO-GO** |
| L2 리폴딩 공정 | 양쪽성·LCST·인공샤페론 | **GO\*** 모델 먼저 |
| L3 제제 열안정 | PEG·부형제·고정화 | **GO** |

**순위:** P1 PEG(모델 효소) → P2/P4 리폴딩 → P3 nanoMIP 후순위.  
**열안정 직관:** L3에서 맞음. L1으로 확장하면 근거 붕괴.  
**키노트:** `05_고분자_적용감사` (최종) · 개관은 `25`/polymer_chaperone_bridge
