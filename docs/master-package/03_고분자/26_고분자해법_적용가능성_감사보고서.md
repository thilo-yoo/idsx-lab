## 2. 제안별 상세 감사

### 2.1 P1 — PEGylation (L3) → **GO**

**지지 근거**
- 치료 단백질 PEG 접합: 용해도, 열·화학 안정, 응집 억제, PK 개선이 반복 보고 (리뷰 다수; 예: Zuma et al. 2022 계열).
- 분자동역학: PEG가 고온에서 소수 코어로의 물 침투를 막아 2차구조 유지에 기여 (insulin/lysozyme 등).
- 다만 **부위·밀도에 따라 Tm·활성이 오르거나 내릴 수 있음** → “PEG = 무조건 안정”은 틀림.

**LSD/ERT 맥락**
- 리소좀 효소 ERT에서 안정성·클리어런스 개선을 위한 접합·제제 전략이 논의·개발되어 왔음 (일반 PEG-단백질 논리 + LSD 나노/제제 리뷰).
- Hunter SoC는 **idursulfase**; CNS 미충족 해결의 주류 첨단은 **BBB 셔틀 ERT(DNL310 등)** 이지 “PEG 단독”이 아님.  
  → PEG는 **제제 안정 도구**이지 CNS 문제의 만능키가 아님.

**IDS 특이 위험**
- M6P 인식 방해 가능 (과도 접목).
- 리소좀 pH에서 활성 유지 필요.
- 변이 대립유전자 세포 구제와 논리 불연속 (보통 **접힌 WT 재조합 효소** 대상).

**학부 실행**
- 상업 rhIDS 확보 난 → **모델 효소**로 방법 확립 후 협력.
- 측정: DSF Tm, 열실활, 활성, DLS — 화공 실험실 친화.

**판정:** L3 **GO**. L1 **NO-GO**. IDSX claim과 **문서 분리** 필수.

---

### 2.2 P2 — 양쪽성 고분자 / 나노샤페론 리폴딩 (L2) → **GO\***

**지지 근거**
- 고전: Rozema–Gellman 계면활성+시클로덱스트린 2단 인공 샤페론.
- PNIPAM·mixed-shell micelle: LCST 이상 hold → 냉각 방출 리폴딩 (다수 보고).
- 고분자 나노샤페론: 모델 효소 활성 회수 고효율 보고 (조건 의존; 고수치 과시 금지).
- 2020–26 리뷰: polymer artificial chaperones 활발.

**한계 (감사 핵심)**
- 성공 사례 대부분이 **소형 모델 단백질** (lysozyme, carbonic anhydrase 등).
- **전장 IDS**(대형, 다수 이황화, N-글리코) 직접 데이터 부족 → 외삽 **COND**.
- **in vitro 희석 리폴딩** ≠ 세포 ERQC/ERAD.
- 생체 투여 경로로서의 독성·분포는 별 문제.

**판정:**  
- 공정 리폴딩·방법론 논문: **GO\***  
- IDS 직접 적용: **CONDITIONAL** (모델 성공 후)  
- 세포 PC 대체: **NO-GO**

---

### 2.3 P3 — nanoMIP (L2 개념) → **CONDITIONAL / 후순위**

**지지 근거**
- PNAS 2024 계열: flexible nanoMIP가 변성 단백질 선택 상호작용·리폴딩 보조, 회수 ~87% 보고.
- 개념적으로 IDSX **표면 에피토프 각인**과 동형.

**감점**
- 템플릿 대량·순도, 전장 IDS 각인 비현실.
- 패치 펩티드 각인 → 전장 모듈 안정 보장은 **미검증 비약**.
- 합성·분석 부담 큼 (학부 1순위 부적합).
- 생체 약리 데이터 빈약.

**판정:** Discussion·2차 개념 파일럿만. **본선 하향**.

---

### 2.4 P4 — LCST 사이클 → **GO\*** (P2와 동일 계열)

PNIPAM LCST ~32 °C 근처로 생리학 온도 근처 조작 가능. 공정 리폴딩 도구로 문헌 강함.  
생체 투여 치료 샤페론으로의 확장은 P2와 동일한 NO-GO 논리.

---

### 2.5 L1 고분자 세포내 샤페론 → **NO-GO** (강화)

| 병목 | 내용 |
|------|------|
| 투과 | ER lumen까지 치료 농도 도달 미확립 |
| 표적 | 정맥 분포 ≠ 효소 폴딩 현장 |
| 면역·축적 | PEG 항체, 폴리머 독성 |
| 농도 | 인공 샤페론·화학 샤페론 모두 고농도 한계 논의 |
| 전략 | IDSX 본선(구조 triage)과 산만·claim 혼선 |

## 3. IDS 특이 외삽 체크리스트

적용 실험 전 필수:

- [ ] 단백질 크기·이황화·도메인 복잡성 고려  
- [ ] N-글리코·**M6P 유지** 여부  
- [ ] pH 4.5–5 활성  
- [ ] FGly 활성형 공정  
- [ ] 변이 vs WT 대상 구분  
- [ ] 모델 효소 → IDS 게이트 통과 기준 사전등록

## 5. 권고 포트폴리오 (감사 후)

```
IDSX 본선 (유지)
  구조 amenability · residual 문헌 · ΔΔG · 소분자 PC 논리

직교 트랙 A — L3 제제 (GO)
  모델 효소 PEG → Tm/활성 맵 → (가능 시) IDS

직교 트랙 B — L2 공정 (GO*)
  RAFT 디블록 또는 LCST 리폴딩 수율 (모델 단백질)

보류
  nanoMIP, 세포내 고분자 샤페론
```

## 6. 근거 유형 (요약)

- PEGylation–stability: 리뷰·MD 다수  
- Artificial chaperone polymers: Rozema–Gellman; PNIPAM/MSPM; nanochaperone 논문·리뷰 (2020–26)  
- nanoMIP chaperones: PNAS 2024 계열  
- MPS II: idursulfase SoC; CNS-ERT는 전달 공학 (PEG 일반론과 구분)  

**메타 한계:** IDS 전용 고분자 데이터는 모델 단백질 대비 **희소** → 외삽 시 보수적 언어 의무.

## 7. 최종 한 줄

**고분자 해법은 실재하며 열·제제·리폴딩에서 강력하다.  
다만 ‘Hunter 변이 세포 구제’로 바로 쓰기에는 근거가 부족하고,  
철저 조사 결과 권고는 “L3/L2 GO + L1 NO-GO + IDSX 본선 유지”로 확정한다.**
