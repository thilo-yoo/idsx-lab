## 1. 세 층 (혼동 방지)

| 층 | 내용 | 고분자 적합성 |
|----|------|----------------|
| **L1** 세포 내 PC | 소분자, genotype-selective | 낮음 |
| **L2** 인공 샤페론 | 리폴딩·응집 억제 공정 | **높음** |
| **L3** 제제·접합체 | PEG, 하이드로겔, 부형제 | **매우 높음** (열안정) |

사용자 직관(“열안정 = 고분자”)은 **L3(및 L2)** 에서 정확하다.  
IDSX 본선 claim의 Tm은 **온도계**이고, L3에서는 Tm 자체가 **목적 함수**가 될 수 있다.

## 2. 방법 카탈로그 (문헌에 실재)

### 2.1 PEGylation (공유 접합)
- 효과: 용해도, **열·화학 안정**, 응집↓, 반감기↑ (위치·밀도 의존; 활성 저하 위험)
- 합성: mPEG-NHS, maleimide–Cys, 클릭; 부위선택 필수
- IDSX: 활성점 원거리 접합 = 6.5Å 철학과 동형

### 2.2 양쪽성 고분자 인공 샤페론
- holdase: 소수성 블록이 중간체 포획 → 응집 억제
- 방출: 희석, 시클로덱스트린, T/pH, 절단 링커
- 합성: **RAFT/ATRP** 디블록 (PEG-b-PNIPAM 등)

### 2.3 온도응답 (PNIPAM, LCST)
- T>LCST: 복합체(억제 응집) → T↓: 방출·리폴딩
- 합성: NIPAM 공중합 조성으로 LCST 튜닝 (화공 반응공학)

### 2.4 nanoMIP (분자 각인 나노입자)
- 템플릿 제거 후 선택 공동 = **에피토프 인식**
- 2024 PNAS 등: flexible nanoMIP as artificial chaperones
- IDSX: 템플릿 = Ser152 **표면 패치 펩티드** (hotspot≠binding site)

### 2.5 쯔비터이온·글리코폴리머
- 강한 hydration, 비특이 흡착↓, 일부 열안정 보고

### 2.6 하이드로겔 고정화
- 효소 고정 → 열·전단 안정, 연속 반응기 (공정 촉매공학)

### 2.7 크로우딩 (PEG, Ficoll)
- 배제 부피 → 접힘 평형 이동 (비공유, 가역)

## 3. 샤페론 기능 ↔ 고분자 합성 맵

| 생체 샤페론 기능 | 고분자·합성 대응 |
|------------------|------------------|
| holdase | 양쪽성 코폴리머, 덴드리머 |
| foldase 보조 | 자극응답 방출 사이클 |
| 선택 인식 | nanoMIP 각인 |
| 안정화 | PEG·쯔비터이온 접합/코팅 |
| 환경 창 | LCST / pH 링커 (= STEP7의 고분자판) |

## 4. 구체 제안 (실행 우선순위)

### P1 — PEG–IDS(또는 모델 술파타제) 열안정 패키지 ★추천
- **층:** L3  
- **합성:** mPEG-NHS/maleimide, DS·Mn 변수  
- **측정:** Tm (DSF), 열실활 t1/2, 활성, DLS  
- **성공 예:** ΔTm ≥ +2 °C 또는 t1/2 ≥ 1.5× (사전등록)  
- **논문 성격:** 제제·생체재료 · PC claim과 분리 가능  

### P2 — RAFT 양쪽성 디블록 인공 샤페론
- **층:** L2  
- **합성:** RAFT, Mn/Đ/LCST 특성화  
- **측정:** 변성 후 활성 회수율  
- **연결:** STEP2 트랩 억제 공정  

### P3 — nanoMIP × 표면 패치
- **층:** L2 개념  
- **템플릿:** IDSX 에피토프 펩티드  
- **연결:** “각인 = 고분자판 에피토프” Discussion  

### P4 — LCST 사이클 최적화
- **층:** L2  
- **합성:** NIPAM 공단량체 조성 DOE
