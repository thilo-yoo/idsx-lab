## 4. IDSX와의 관계

| | 아바타 | IDSX |
|--|--------|------|
| 질환 공간 | LSD (Krabbe 등 공개 협업) | Hunter / IDS (MPS II) |
| 모달리티 | AAV gene therapy | PC·구조 amenability triage |
| 강점 | 전달·생산·AI 캡시드 | 구조 필터·genotype 의사결정 층 |
| 공통 | 희귀 LSD, CNS 난제, AI/계산, 정밀 환자 층위 | |

**한 줄 포지션**  
아바타가 “어떻게 전달·생산할까”라면, IDSX는 “누구에게 어떤 안정화 논리를 우선할까”다. **같은 LSD 전장, 직교 무기.**

## 5. IDSX에 반영할 내용 (체크리스트)

- [ ] **R1** LSD 공통 서론 (Hunter ⊂ LSD, CNS 공유 과제)  
- [ ] **R2** 모달리티 직교 figure (ERT / PC / AAV)  
- [ ] **R3** CNS 전달 = 별 게이트 (아바타 과제와 역할 분리)  
- [ ] **R4** CMC·화공 언어로 S1–S8·PFI 재포장  
- [ ] **R5** genotype decision layer 강조 (희귀·이질 변이)  
- [ ] **R6** 국책 키워드 매핑 1페이지 (AI, 바이오공정, 유전자치료, 희귀)  
- [ ] **R7** 면담 문장 고정 (본 덱 §10)  
- [ ] **R8** 기존 분석 PPT의 대표명 등 팩트 → 공식 정보로 교정
