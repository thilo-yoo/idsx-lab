# 샤페론 외 치료·개입 축 — MPS II / LSD 맵과 IDSX 반영

**버전:** 2026-07-17  
**지위:** 공개 문헌·파이프라인 개요 기반 **지형 노트**. 임상 처방·특정 약 추천 아님.  
**관련 덱:** `IDSX_NonChaperone_Modalities_Landscape.key`

## 1. 왜 논의하는가

1. IDSX **Hard EXCLUDE** (null, 촉매 kill 등) → PC 비대상 → 다른 모달리티가 본선  
2. **신경형** MPS II 미충족 = BBB/CNS → IV ERT·소분자 PC만으로 부족할 수 있음  
3. ERT·GT·HSCT 등이 임상·규제 현실에서 이미 움직임  
4. 기업(아바타 등)·국책 대화 = AAV/LSD 축과의 **직교 언어** 필요  
5. 공정 사고: 같은 효소 부족이라도 **공급원·전달·현장** 해법이 다름

## 2. 모달리티 카탈로그

### 2.1 지지요법
- 기도·심장·정형외과·재활·청력·다학제 케어  
- 모든 질환 수정 치료와 **병행**  

### 2.2 IV ERT (표준)
- **Idursulfase** 주 1회 IV — SoC  
- 체성 증상 개선 기여 보고  
- **BBB 제한** → 신경형 미충족  
- 평생 투여, 비용, 면역원성  

### 2.3 CNS 지향 ERT
- **BBB-penetrant ERT** (예: TfR 셔틀 융합 IDS — tividenofusp alfa/DNL310 등; **승인 상태는 시점 재확인**)  
- **IT/ICV ERT** — 척수강·뇌실 직접 투여 (시술 부담)  

### 2.4 HSCT
- 공여자 유래 세포가 I2S 공급원 역할 가설  
- MPS II에서 효용·시기 **논쟁**  
- GVHD·전처치·공여자 문제  

### 2.5 In vivo AAV gene therapy
- 예: **RGX-121** (clemidsogene lanparvovec) — AAV9로 IDS를 CNS 경로 전달 설계  
- 1회 투여·지속 발현 목표; CSF 바이오마커 보고  
- 규제 경로 **유동적** (BLA 리뷰 연장·거부 보도 등 — 최신 확인 필수)  

### 2.6 HSC gene therapy
- 체외 렌티바이러스 등으로 HSC에 IDS (± 뇌 표적 태그)  
- 전임상~초기 임상  

### 2.7 Genome editing
- base/prime editing 등 — 원천 교정 잠재, 전달·안전 초기  

### 2.8 SRT / readthrough
- 기질 감소: MPS II 본선 아님 (다른 LSD에서 더 확립)  
- stop readthrough: nonsense 일부, MPS II 임상 제한적  

### 2.9 (참고) 약리 샤페론 / 바인더 — IDSX 본선
- 접힘·ER 통과 가능 미스센스에 **선택적**  
- 본 문서의 “외” 축과 **직교**

## 4. S1–S8에 꽂기

| 단 | 비-PC 주 개입 |
|----|----------------|
| S1 | GT·편집 (유전자 공급) |

| S6 | BBB-ERT, CNS-GT, HSCT |
| S7 | 공급 효소 활성 유지 |
| S8 | 모든 축의 종점 KPI |

## 5. 상위 라우팅 트리 (스케치)

```
null/fs? → ERT / GT / HSCT
촉매 kill? → ERT / GT
신경형? → CNS 전달 모달리티 상위 게이트
미스센스 + residual 가능? → PC triage (+ 현실적 ERT 병행)
glyco/PTM 파괴? → ERT/GT 우선
```

**IDSX amenability = 네 번째 분기의 구조 필터.**

## 6. IDSX 반영 체크리스트

- [ ] 논문/포털에 **모달리티 스펙트럼 figure**  
- [ ] EXCLUDE 행에 **라우팅 라벨** (GT/ERT 축) — 치료 추천 문구 없이  
- [ ] CNS 미충족 **non-claim 박스**  
- [ ] 직교 병용 가설 1단락 (과신 금지)  
- [ ] 아바타 등 기업 대화: 전달 vs triage 분리  
- [ ] 파이프라인 스냅샷 **날짜 스탬프** (규제 변동)
