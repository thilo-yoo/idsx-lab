# 비전문가·환자 가족·비전공 동료용 언어 가이드

**목적:** 정확한 비유 + **과신 금지**. 의료 조언 아님.

---

## 1. 30초 설명

우리 몸 세포의 “재활용 공장”(리소좀)에서 IDS라는 효소가 특정 당을 치웁니다.  
헌터 증후군은 이 효소가 부족해 당이 쌓이는 병입니다.  

어떤 유전자 변화는 효소의 **칼날(반응 자리)** 을 망가뜨리고,  
어떤 변화는 효소가 **공장에서 제대로 조립·배달되지 못하게** 합니다.  

IDSX는 “약을 이미 만들었다”는 연구가 아니라,  
**어떤 종류의 변화에 어떤 접근을 시도할 가치가 있는지**  
효소의 입체 구조를 보고 분류하는 지도 작업입니다.

## 2. 추천 비유

| 개념 | 추천 비유 | 비유 한계 (꼭 말하기) |
|------|-----------|------------------------|
| 리소좀 | 재활용 공장 | 공장은 하나만이 아님 |
| IDS | 특정 쓰레기를 자르는 가위 | 가위 외 역할도 있음 |
| 미스센스 | 설계도 한 글자 변경 | 모든 오타가 같은 결과 아님 |
| Ser152 코어 | 내부 나사/핀 | 나사가 보이진 않음 |
| ERQC | 품질검사 | 서열을 읽는 게 아님 |
| ERAD | 불량 폐기 | 항상 전부 폐기 아님 |
| 샤페론 | 조립 보조 지지대 | 만능 치료 아님 |
| 에피토프 | 밖에서 잡는 손잡이 | 손잡이≠고장 나사 |
| TIER1 | “먼저 살펴볼 후보” | 치료 확정 목록 아님 |

## 3. 금지 문장 → 교정

| 금지 | 교정 |
|------|------|
| 이 변이는 치료됩니다 | 구조상 **이런 논리로 연구가 가능한지** 보는 단계입니다 |
| 암브록솔을 쓰면 됩니다 | 다른 병에서 배운 **조건**을 참고할 뿐입니다 |
| AI가 구조를 고쳤습니다 | AI는 **그림을 그린 것**이고, 안정·치료와는 다릅니다 |
| TIER1 환자는 약 대상 | TIER1은 **연구 우선순위 표시**입니다 |
| 152번에 약이 붙습니다 | 152는 안쪽에 있어, **옆 표면**을 잡는 전략을 생각합니다 |

## 4. Q&A 짧은 답

**Q. 지금 약이 있나요?**  
A. 이 프로젝트는 약을 만든 단계가 아닙니다. 표준 치료는 의료진과.

**Q. 우리 아이 변이가 지도에 있으면?**  
A. 연구 분류와 개인 치료 결정은 다릅니다. 담당 의료진 상담이 필요합니다.

**Q. 왜 화학공학?**  
A. 효소 분자 수를 **공장 유량**처럼 보고, 어디서 새는지 가릅니다.
