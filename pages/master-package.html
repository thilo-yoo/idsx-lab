화학공학 렌즈 (이 패키지의 정체성)

  • 물질수지 — 기능 IDS 분자 유량 (합성 − ERAD − 미배달)
  • 단위공정 — ER → QC → 골지/M6P → 리소좀 반응기
  • 결함 모드 — 촉매 kill / 코어 불안정 / 포켓 인접 / PTM
  • 열역학 — ΔΔG = 에너지 수지 · AF 그림 ≠ 에너지 상태
  • KPI 계층 — 최종 = competence · Tm = 온도계
  • 스케일업 윤리 — 게이트 없는 대량 설계 금지

자문용 세 문장

1. IDS 미스센스를 독립 라벨이 아니라 기능 구조를 떠받치는 네트워크의 교란으로 읽는다.
2. Ser152는 촉매 자리가 아니라 코어 모듈이며, 개입은 인접 표면 패치를 향한다.
3. 성공은 결합이 아니라 catalytic competence이며, 현재는 구제 증명 전이다.
